# クライアントアプリケーション

## Node.JSのパッケージをインストールする

```
npm i
```

## アプリケーションのビルド

```
npm run webpack
```

## ファイルの変更を検知してビルド

```
npm run webpack -- -w
```

## ビルドされたファイル群の削除

```
npm run clean
```

## テストコードの実行

```
npm t
```
